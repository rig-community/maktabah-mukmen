Cara Install di Android

Empat langkah mudah dan aplikasi siap digunakan.

Unduh File APK

Tekan tombol Download di atas. File APK akan tersimpan di folder Unduhan perangkat Anda.

Izinkan Sumber Tidak Dikenal

Masuk ke Pengaturan → Keamanan (atau Privasi) → aktifkan "Instal aplikasi dari sumber tidak dikenal".

Buka File APK

Buka aplikasi File Manager, cari file maktabah_mukmen.apk, lalu ketuk untuk memulai proses instalasi.

Install & Mulai Membaca

Tekan "Instal" dan tunggu selesai. Buka aplikasi, pilih kategori kitab, dan mulai perjalanan ilmu Anda.
